# Example Mods
These are example mods that can be built using the modding SDK.

* **Expanded Creative Mod**: This mod adds even more items and blocks to the Creative Inventory. It was originally by [@Bigjango13](https://github.com/bigjango13).
* **Chat Commands Mod**: This mod makes a chat message starting with `\` handled by the MCPI API.
* **Recipes Mod**: This mod adds custom recipes.
* **Custom Block Mod**: This mod adds a custom block.
